!  It's recommended to run script `free_ports.sh` first.
-> `build.sh` - compiles and builds project.
-> `runTestsThroughBankTests.sh` - runs `BankTests.java` and returns exit code (0 on success, 1 on error).
-> `runTestsThroughJUnit.sh` - runs tests using standard JUnit methods
-> `startServer.sh` - starts the server
-> `runClient.sh` - runs the client side
-> `clean.sh` - clean ".class"-files and removes "out" directory
